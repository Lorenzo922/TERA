🚀 TERA Engine v0.1.3

Motor simples de execução de scripts com interface gráfica

Baixar TERA Engine

🆕 Novidades da versão 0.1.3

  • 🪟 TeraPlus = Window(Altura.Largura) agora cria uma janela interativa
  • 🏷️ TeraPlus.AddLabel("Texto") adiciona um texto à janela
  • 🔘 TeraPlus.AddButton("Texto") cria um botão funcional
  • 🛠️ Melhorias visuais e suporte a múltiplos componentes

🧪 Exemplo de uso

Import = TeraPlus
TeraPlus = Window(300.500)
TeraPlus.AddLabel("Bem-vindo ao TERA Engine!")
TeraPlus.AddButton("Clique aqui")
Print("Janela criada com conteúdo!")
